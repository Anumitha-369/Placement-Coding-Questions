# First Odd Length Word

## Problem Statement

Write a Python program to read `N` words and print the **first word with an odd number of characters**.

If no word has an odd length, print:

```
Better luck next time
```

### Input Format
- First line: Integer `N` (number of words)
- Second line: `N` space-separated words

### Output Format
- Print the first word with odd length.
- Otherwise, print `Better luck next time`.

### Example 1

Input:
```
5
Hello Good morning Welcome you
```

Output:
```
Hello
```

### Example 2

Input:
```
3
Go to hell
```

Output:
```
Better luck next time
```
